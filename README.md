# Deliverawr

### A complete ordering system for restaurants

#### Various Components : 
- Landing Page
- Order Placement System
- Payment System
- User Registeration System

#### Languages Used : 
- Python
- HTML
- CSS
- Javascript

#### Framkeworks Used : 
- Django
- Stripe
- Django-Storages
- Raven
- PostgresSQL
- Docker

#### Services Used :
- Mailchimp : Mail (Order Confirmation, User Validation, Etc)
- Sentry : Error handnling and managment
- Twilio : SMS 
- HeapAnalytics : Anaytics

#### Hosted on : 
- Website : Heroku
- Static Media : AWS-S3
